British Airways Airline Review Dashboard using Tableau

📊 Project Overview

This project analyzes customer reviews of British Airways using an Excel dataset and presents insights through an interactive Tableau dashboard. The goal was to explore service quality trends across time, geography, aircraft type, and traveler profiles.

🔑 Key Features

    Interactive Dashboard: Filters by traveler type, seat class, continent, and aircraft.

    Trend Analysis: Cabin staff service ratings tracked monthly from 2016–2023.

    Geographic Insights: Country‑level visualization of service ratings.

    Aircraft Comparison: Average ratings and review volumes across different aircraft models.

    Service Metrics: Food & beverages, entertainment, seat comfort, ground service, and value for money.

🛠️ Tools & Skills

    Tableau (visualization, mapping, trend analysis)

    Excel (data cleaning, preparation, integration)

    Data storytelling (turning raw feedback into actionable insights)

🚀 Impact

This project demonstrates how customer feedback can be transformed into structured insights that airlines can use to improve service quality, optimize fleet performance, and enhance passenger experience.
